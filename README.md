# 教室白板系统

基于手势控制的全屏透明批注系统。Python + MediaPipe 做手势识别，PyQt5 做透明全屏画布。

## 架构

```
┌─────────────────────────────────────────────┐
│           PyQt5 透明全屏窗口（前端）           │
│  - 系统级置顶透明窗口                          │
│  - QPainter 画线（GPU 加速）                   │
│  - 摄像头预览（右下角）                         │
│  - 键盘快捷键                                 │
└────────────▲────────────────────────────────┘
             │ queue.Queue（手势数据）
┌────────────┴────────────────────────────────┐
│         MediaPipe 引擎（后端线程）              │
│  - 摄像头读取（640x480）                       │
│  - 手部关键点检测                              │
│  - 手势识别（防抖+松开缓冲）                    │
└─────────────────────────────────────────────┘
```

## 功能

### 手势

| 手势 | 功能 |
|------|------|
| 食指+拇指捏合 | 画画/批注 |
| 五指全部张开 | 橡皮擦 |
| 双手拍手 | 切换颜色 |
| 张开↔握拳两次 | 清空 |
| 食指+小指 | 切换白板/批注模式 |

### 键盘

| 键 | 功能 |
|------|------|
| `E` | 切换橡皮擦/画笔 |
| `N` / `C` | 清空 |
| `Z` | 撤销 |
| `A` | 切换白板/批注模式 |
| `Q` | 退出 |

## 安装

```bash
pip install mediapipe>=0.10.0 opencv-python>=4.5.0 numpy>=1.20.0 PyQt5>=5.15.0 mss>=9.0.0 Pillow>=9.0.0
```

## 运行

```bash
python main.py
```

## 项目结构

```
whiteboard/
├── main.py                    # 启动入口（连接引擎和窗口）
├── core/
│   ├── __init__.py
│   ├── hand_detector.py       # 手部检测（MediaPipe）
│   ├── gesture_detector.py    # 手势识别（防抖+松开缓冲+相对阈值）
│   ├── gesture_engine.py      # 手势引擎（后台线程，纯计算）
│   ├── overlay_window.py      # PyQt5 透明全屏窗口（前端渲染）
│   ├── canvas.py              # OpenCV 画布（旧版，保留备用）
│   ├── annotation_overlay.py  # 屏幕截图批注（旧版，保留备用）
│   └── whiteboard_main.py     # OpenCV 主控制器（旧版，保留备用）
├── models/
│   └── hand_landmarker.task   # MediaPipe 模型
├── README.md
└── requirements.txt
```

## 对比旧版

| | 旧版（OpenCV） | 新版（PyQt5） |
|--|---|---|
| 窗口 | OpenCV 窗口（不透明） | 系统级透明窗口 |
| 背景 | 白色画布 / 截图 | 真正的屏幕 |
| 渲染 | CPU（OpenCV） | GPU（Qt QPainter） |
| 架构 | 单线程同步 | 前后端分离（多线程） |
| 批注 | 截图上画（低帧率） | 直接屏幕画（实时） |
